# 📄 Sistema de Gerenciamento de Contratos

Sistema web desenvolvido com **Django** para gerenciamento de contratos e usuários, permitindo controle, organização e visualização eficiente das informações.


## 🚀 Sobre o Projeto

O sistema foi desenvolvido com o objetivo de facilitar o gerenciamento de contratos e usuários dentro de uma organização, oferecendo:

- Cadastro e gerenciamento de usuários
- Controle de permissões por grupo
- Cadastro e consulta de contratos
- Filtros por cidade, bairro e rua
- Interface administrativa personalizada
- Paginação e organização de dados
- Sistema de status com badges visuais

## 🛠️ Tecnologias Utilizadas

- **Backend:** Python + Django  
- **Frontend:** HTML, CSS, Bootstrap  
- **Banco de Dados:** PostgreSQL / MySQL / SQL Server (ajustar conforme uso)  
- **JavaScript:** Vanilla JS para interações dinâmicas
